# BestPerfume Catalogue

This application is a perfume catalogue for BestPerfume, a Singaporean perfume retailer selling cheap and concentrated resemblances of designer perfumes. The application is designed to ease browsing and searching for a suitable perfume compared to using the BestPerfume website.

Currently, the application only supports Windows and Android. Head over to Releases and download the appropriate file based on your device.
- For Windows, download the .zip file, extract it, and run the .exe file within. You may wish to right-click the .exe file and pin to the Start menu for ease of access.
- For Android, simply download and run the .apk file. You may scan the app when prompted if you wish, but this is not necessary if you trust Jamie.
